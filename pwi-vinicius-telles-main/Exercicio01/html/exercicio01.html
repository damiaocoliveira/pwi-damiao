Tags básicas para a criação de página HTML e CSS

O que é HTML?

Hypertext Markup Language,
ela permite que os desenvolvedores desenhem como serão exibidos pelo navegador os elementos da página, como texto, hiperlinks e arquivos multimídia

Para criar um título é necessário utilizar a tag <h1> essa tag possui seis níveis de títulos que vão do h1 ao h6, sendo o h1 o maior tíitulo e h6 o menor

Para criar um parágrafo é necessário utilizar a tag <p>

Para quebra de linha do texto é usado tag <br>

Para a criação de comentários podemos utilizar o comando <!-- -->

Comandos básicos CSS

O que é CSS?

linguagem de marcação, amplamente utilizada com HTML ou XHTML, e representa diversas possibilidades para a formatação. Com o Css podemos estilizar e toda a página do nosso site
